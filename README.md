# cellnet

`Simple mesh network`

####How to start:


1. install python 3
2. go to project dir
3. ```$: pip install --upgrade virtualenv ```
4. ```$: virtualenv -p python3 env ```
5. ```$: source env/bin/activate```
6. ```$: pip install -r requirements.txt ```
7. mark _env/lib/python3.5/site-packages_ as source root
8. ```$: python src/main.py ```
